# 모의주식 프로그램

간단한 주식 거래 시뮬레이션 프로그램입니다.

## 기능

- 주식 거래 (매수/매도)
- 계좌 잔고 확인
- 보유 주식 현황 확인
- 계좌 입금

## 실행 방법

### 방법 1: 소스 코드 직접 컴파일

1. Java 개발 환경 준비 (JDK 17 이상 권장)

2. 프로젝트 클론 또는 다운로드

```bash
git clone [repository-url]
cd java-stock
```

3. 컴파일

```bash
javac -d bin src/*.java
```

4. 실행

```bash
java -cp bin Main
```

### 방법 2: JAR 파일 직접 실행

1. result 폴더의 stock-trading.jar 파일 다운로드

2. 실행

```bash
java -jar stock-trading.jar
```

## 고객사 요구사항

1-1. 간단한 서비스 설명 - 모의주식프로그램(금융 API 활용)으로 매수와 매도,내정보 확인 기능이 있습니다.

1-2. 기능 요구사항 명세

```text
System.out.println("모의주식 프로그램에 오신것을 환영합니다");
기능을 선택하세요 : 1. 거래 2. 내정보 3. 입금
----<2번 입력 후(내정보 확인 기능)>----
현금 : 0원
보유 종목: 없음
----<3번 입력 후(입금 기능)>----
얼마를 입금하시겠습니까? : 1000000
----<입금후>----
보유 현금 : 1,000,000원
입금이 완료되었습니다.

----<종목코드 입력 후>----
```
